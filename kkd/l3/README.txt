uruchomienie programu:
nalezy być terminalem w folderze z plikami lzw.py oraz methods.py oraz in_out.py

python lzw.py <opcje>

opcje:
  -i <input file> (argument wymagany)
  -o <output file> (argument wymagany)
  -e zakodowanie / encoding
  -d odkodowanie / decoding

  (opcje metody)
  --gamma
  --delta
  --fibb
  --omega (domyślna)
